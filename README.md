A Python-based Personal Expense Tracker with a graphical user interface built using Tkinter.
The application allows users to add daily expenses with date, category, and amount.
It stores expense data in a CSV file for easy and lightweight persistence.
Users can view monthly expense totals and category-wise reports with a grand total.
Input validation is implemented to ensure correct date format and numeric values.
This project demonstrates Python GUI development, file handling, and data procduct.
